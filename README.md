# پروژه بامارام

این یک پروژه با بک‌اند پایتون (FastAPI) و فرانت‌اند ری‌اکت است.

## ساختار پروژه

```
bamaram/
├── backend/                    # پوشه مربوط به بک‌اند
│   ├── app/                    # کدهای اپلیکیشن پایتون
│   │   ├── crud/               # عملیات CRUD
│   │   ├── database/           # تنظیمات پایگاه داده
│   │   ├── models/             # مدل‌های SQLAlchemy
│   │   ├── routers/            # روترهای API
│   │   ├── schemas/            # طرح‌های Pydantic
│   │   ├── services/           # سرویس‌های کاربردی
│   │   └── main.py             # نقطه ورود اصلی
│   ├── venv/                   # محیط مجازی پایتون
│   ├── requirements.txt        # لیست وابستگی‌های پایتون
│   ├── bamaram.db              # فایل دیتابیس SQLite
│   └── .env                    # فایل متغیرهای محیطی
│
├── frontend/                   # پوشه مربوط به فرانت‌اند (React)
│   ├── src/                    # کدهای React
│   ├── node_modules/           # وابستگی‌های نود
│   ├── package.json            # تنظیمات و وابستگی‌های React
│   ├── public/                 # فایل‌های استاتیک
│   └── build/                  # پوشه‌ی فایل‌های ساخته‌شده برای دیپلوی
```
